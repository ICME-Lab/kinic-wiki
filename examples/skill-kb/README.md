# Skill KB Example

Golden sample package for the Skill Registry quickstart.
Read [`../../docs/QUICKSTART_SKILL_KB.md`](../../docs/QUICKSTART_SKILL_KB.md) first, then use this directory as the source package and run evidence fixture.

Files:

- `skills/legal-review/manifest.md`: registry metadata.
- `skills/legal-review/SKILL.md`: agent-facing skill entrypoint.
- `skills/legal-review/provenance.md`: source and authorship context.
- `skills/legal-review/evals.md`: lightweight evaluation notes.
- `runs/legal-review-success.md`: example run evidence.
